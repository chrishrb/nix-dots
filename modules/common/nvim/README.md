## Getting started

```bash
nix run github:chrishrb/nix-nvim
```

## Setup Copilot

1. Enable `ai` in `flake.nix`
2. Start `chrisNvim` and run `:Copilot auth`
3. Have fun using copilot in cmp and `:CopilotChat`, ..
